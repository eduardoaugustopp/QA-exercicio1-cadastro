Tabela de Decisão para o Teste de Cadastro

Cenário Comportamento Esperado Resultado Esperado
Campos Obrigatórios Os campos obrigatórios devem ser preenchidos corretamente.
- Campos preenchidos corretamente Cadastro bem-sucedido Cadastro redireciona para a tela de login
- Campos em branco Mensagem de erro é exibida Erro: Campos obrigatórios faltando
Validação de Senha
- Senha com menos de 8 caracteres Mensagem de erro é exibida Erro: Senha muito curta
- Senha com 8 ou mais caracteres Senha aceita Continuar com o processo
Confirmação de Senha
- Senha e confirmação não correspondem Mensagem de erro é exibida Erro: Confirmação de senha não corresponde
- Senha e confirmação correspondem Senha confirmada Continuar com o processo
Cadastro Bem-Sucedido
- Campos preenchidos corretamente Cadastro bem-sucedido Redirecionar para a tela de login
Mensagem de Erro
- Situação de erro simulada Mensagem de erro apropriada exibida Feedback apropriado para o usuário
